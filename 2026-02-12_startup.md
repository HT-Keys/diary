# HT-Keys 始動

これまで個人開発で進めていた自作キーボードをちゃんとプロジェクト化して管理すべく，「HT-Keys」プロジェクトを立ち上げた．  
それに伴って GitHub Organization を作成し，整備を開始した．  
cf. <https://github.com/HT-Keys>

また，アイコンも作成した．キースイッチとキーキャップを正面から見た様子を図案化したものとしたので，キーボード関連のプロジェクトであることが一目で分かるデザインと出来ていそう．

加えて，各種案内やドキュメントの整備のために Organization 名前と同名の Repository を作成．また，活動記録を残すためにこの Repository も作成した．  
cf. <https://github.com/HT-Keys/HT-Keys>, <https://github.com/HT-Keys/diary>
